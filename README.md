# Week3_model_optimization
# Customer Churn Prediction using Machine Learning

##  Project Overview
This project focuses on predicting **customer churn** using machine learning models. Churn prediction helps businesses identify customers who are likely to leave, enabling proactive retention strategies.

The project implements and compares multiple models, including **Random Forest** and **XGBoost**, with hyperparameter tuning to achieve optimal performance.

---

##  Objectives
- Analyze customer data to identify churn patterns  
- Build and evaluate machine learning models  
- Improve performance using hyperparameter tuning  
- Identify key features influencing churn  
- Save the best-performing model for deployment  

---

##  Technologies Used
- Python  
- NumPy, Pandas  
- Matplotlib, Seaborn  
- Scikit-learn  
- XGBoost
